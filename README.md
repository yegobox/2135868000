# yegoMemo
YegoMemo is a simple app to manage your memos in one place. You can write, edit and delete memos when the're not still needed.

## Features
 - Create Memo
 - Update Memo
 - Delete Memo
 - Backup memos
 - Memos Sync


## Libraries used
 - Isar (local storage)
 - Pocketbase (cloud storage)
 - stock (databases synching)
